# Assignment-1 
 Overview   
This project focuses on implementing and analyzing fundamental algorithmic paradigms used in real-world problem solving. The assignment covers three major approaches:
Divide and Conquer
Greedy Algorithms
Dynamic Programming
The goal is to understand how different algorithms behave, compare their efficiency, and analyze their scalability using performance metrics and visualizations.
 Objectives
Implement core algorithms using Python
Analyze execution time and performance
Compare different algorithmic strategies
Solve optimization and sorting problems
Visualize results using graphs
🧠 Algorithms Covered
🔹 Divide and Conquer
Merge Sort
Quick Sort
🔹 Greedy Algorithms
Activity Selection
Fractional Knapsack
🔹 Dynamic Programming
0/1 Knapsack
Fibonacci (DP approach)
 Performance Analysis
Execution time measured using Python time module
Memory usage analyzed using memory_profiler
Graphs plotted using matplotlib
Comparison done for different input sizes
